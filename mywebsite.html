Tinycompiler16

A Python-based compiler project that translates code written in a simple high-level language into custom 16-bit Machine code. The project successfully implements core compiler phases including lexical analysis (tokenization), parsing (syntax tree construction), Abstract Syntax Tree (AST) construction, and binary code generation to simulate low-level instruction processing. This project is particularly useful for learning the internal workings of compilers and understanding instruction encoding.

Tools & Technologies: Python (Programming Language), Regex (Regular Expressions for lexical analysis), Custom ISA (Custom Instruction Set Architecture defined for the 16-bit machine).

The application provides a structured and engaging environment for young learners to explore foundational concepts in Science, Mathematics, and General Awareness through interactive and stimulating puzzles.
